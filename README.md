EMI Patternizer
================

[![Modrinth](https://img.shields.io/modrinth/dt/emi-patternizer?logo=modrinth)](https://modrinth.com/mod/emi-patternizer)
[![CurseForge](https://img.shields.io/curseforge/dt/1480670?logo=curseforge)](https://curseforge.com/minecraft/mc-mods/emi-patternizer)

Create hundreds of patterns with a single key press.  
This mod lets you generate Applied Energistics 2 (AE2) patterns — crafting, processing, etc. — directly from any recipe you select in the EMI recipe tree. The mod also remembers which items have already been encoded and avoids creating duplicate patterns for the same item.

**Usage**  
1. Bind a key for the mod (default: `N`).
2. Select the item you want to encode patterns for — e.g., *Energizing Rod (Nitro)* from *Powah*.
3. Expand the recipe tree in EMI and switch it to **Crafting** mode.
4. Open the *Pattern Encoding Terminal* and make sure there are enough blank patterns in the blank-pattern slot.
5. Press the key you bound to start encoding.

**Tips**  
1. The mod is in beta! please report bugs via the issue tracker.
2. I currently don't have time to port the mod to older Minecraft versions or to other mod loaders.
3. Open the *Patten Access Terminal* will clear the mod's remembered items. Reading items from that terminal is planned for the next release.(partially implemented in 0.2)
